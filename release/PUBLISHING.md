# PatentCraft / 专利工艺

PatentCraft 是一个 Fabric 1.21.1 服务器玩法 Mod。

受保护物品第一次被玩家制作时会自动登记专利，第一位制作者会获得对应的专利书并触发全服播报。其他玩家制作出的受保护物品默认无法使用，需要用对应专利书授权后才能正常使用。

## 功能

- 首次制作受保护物品时自动登记专利。
- 第一制作者获得专利书。
- 未授权的受保护物品无法使用、放置、攻击、挖掘或正常装备。
- 受保护物品可以和对应专利书合成授权。
- 每本专利书只能使用一次。
- 玩家可以手持普通书右键讲台打开专利登记册。
- 专利拥有者可以制作专利书、授权其他玩家制作专利书、收回授权。
- 专利拥有者可以开源专利，让所有人都能使用该物品。
- 创造模式管理员可以使用专利工作台调整专利白名单。
- 支持普通合成和锻造台下界合金升级的专利登记。

## 默认保护物品

默认保护钻石工具、钻石武器、钻石盔甲、下界合金工具、下界合金武器、下界合金盔甲和重锤。

管理员可以在游戏内使用专利工作台调整白名单。

## 安装

1. 安装 Fabric Loader。
2. 安装 Fabric API。
3. 将 `patentcraft-1.0.0-beta.jar` 放入 `mods` 文件夹。
4. 客户端和服务端都需要安装。
5. 使用 Minecraft 1.21.1 启动游戏。

## 依赖

- Minecraft 1.21.1
- Fabric Loader 0.16.0 或更高
- Fabric API
- Java 21
